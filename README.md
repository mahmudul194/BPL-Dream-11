# 🏏 BPL Dream 11

A modern and interactive **fantasy cricket team builder** built with React. This application allows users to create their own dream team by selecting players, managing coins, and tracking selections in real time.

## 📌 Project Overview

**BPL Dream 11** is a responsive web application inspired by fantasy sports platforms. Users can browse available players, select their preferred squad, and manage their budget efficiently.

This project demonstrates front-end development skills including state management, component-based architecture, and UI design implementation.

---

## ⚙️ Tech Stack

- **React.js** – Frontend library for building UI  
- **Vite** – Fast development build tool  
- **Tailwind CSS** – Utility-first CSS framework  
- **DaisyUI** – Tailwind component library  
- **JavaScript (ES6+)** – Core programming language  
- **React Toastify** – For notifications and alerts  
- **JSON Data + Fetch API** – For dynamic player data  

---

## ✨ Features

- 🧑‍🤝‍🧑 **Player Selection System**
  - Add players to your team
  - Prevent duplicate selections
  - Limit player count

- 💰 **Coin Management**
  - Claim free credits
  - Deduct coins when selecting players
  - Prevent selection if coins are insufficient

- 🔄 **Dynamic UI Updates**
  - Switch between available and selected players
  - Real-time count updates

- 🔔 **Notifications**
  - Success & error messages using React Toastify

- 📱 **Responsive Design**
  - Works on mobile, tablet, and desktop devices

---


---

## 🧠 How It Works

- Player data is fetched from a local JSON file.
- Users can select players based on available coins.
- State is managed using React hooks (`useState`, `use`).
- UI updates dynamically based on user actions.
